# 运算

## 1. 按位与：用来保留某些位/清零某些位

```
1 & x = x   保留
0 & x = 0   清零
```

## 2. 按位或：orL：把 某些位置1

```
1 | x = 1   强制置1
0 | x = x   保留
```

## 3. 按位异或：把某些位取反

```
1 ^ x = ~x  取反
0 ^ x = x   保留
```

## 4. not按位取反

十六进制取反：直接每一位用F减去就行

# 移位指令

1. shl/sal:左移。最高位被移出，进入CF，低位布0。相当于×2（不过注意溢出）
2. shr：逻辑右移，适合无符号数。最低为移除，进入CF，最高位补0
3. sar：算数右移，适合有符号数。最高位补原来的符号位，最低为移入CF
