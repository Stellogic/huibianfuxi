## 1. 选择题精简背诵

### 1.1 ISA 与架构

- `ISA` = Instruction Set Architecture，指令集体系结构。
- x86 / x86-64：典型 `CISC`，指令格式多，指令长度可变，很多指令可直接访问内存。
- ARM / AArch64 / 鲲鹏：典型 `RISC`，常见固定长度指令，Load/Store 风格明显。
- 鲲鹏相关只按选择题准备：大题不考鲲鹏编程。

### 1.2 数据类型和大小

| 名称 | 大小 |
|---|---:|
| byte | 1 字节 |
| word | 2 字节 |
| dword / long | 4 字节 |
| qword / quad | 8 字节 |
| 128 位双四字 | 16 字节 |

GAS 伪指令：

| GAS | 大小 |
|---|---:|
| `.byte` | 1 |
| `.word` | 2 |
| `.long` | 4 |
| `.quad` | 8 |
| `.ascii "abc"` | 不补 0 |
| `.asciz "abc"` / `.string "abc"` | 补 `\0` |
| `.space n` | 分配 n 字节 |
| `.align n` | 按边界对齐 |


**定义数据看 GAS 伪指令；访问内存看 Intel 大小说明。**

| 你想干什么 | 用什么 | 例子 |
|---|---|---|
| 在 `.data` 里放 1 字节 | `.byte` | `a: .byte 1` |
| 在 `.data` 里放 2 字节 | `.word` | `b: .word 2` |
| 在 `.data` 里放 4 字节 | `.long` | `c: .long 3` |
| 在 `.data` 里放 8 字节 | `.quad` | `d: .quad 4` |
| 访问内存 1 字节 | `byte ptr` | `mov al, byte ptr [rdi]` |
| 访问内存 2 字节 | `word ptr` | `mov ax, word ptr [rdi]` |
| 访问内存 4 字节 | `dword ptr` | `mov eax, dword ptr [rdi]` |
| 访问内存 8 字节 | `qword ptr` | `mov rax, qword ptr [rdi]` |
### 1.3 有符号 / 无符号范围

```text
n 位无符号：0 ~ 2^n - 1
n 位有符号补码：-2^(n-1) ~ 2^(n-1)-1
```

常背：

- 8 位有符号：`-128 ~ 127`
- 16 位有符号：`-32768 ~ 32767`
- 32 位有符号：`-2^31 ~ 2^31-1`

负数补码：

```text
绝对值二进制 -> 按位取反 -> 加 1
```

快速判断：

```text
8 位 0xFF = 无符号 255 = 有符号 -1
8 位 0x80 = 无符号 128 = 有符号 -128
```

### 1.4 标识符合法性

常规判断口径：

- 可以由字母、数字、下划线等组成。
- 一般不能以数字开头。
- 不能和指令名、寄存器名、伪指令等冲突。
- 区分题目给的是 C 标识符规则还是汇编标识符规则。

### 1.5 汇编指令常见错误

必背禁忌：

1. 不能内存到内存直接传送。

```asm
# 错
mov dword ptr [rdi], dword ptr [rsi]

# 对
mov eax, dword ptr [rsi]
mov dword ptr [rdi], eax
```

2. 立即数不能作目的操作数。

```asm
# 错
add 1, eax

# 对
add eax, 1
```

3. 内存操作数常要写类型。

```asm
mov dword ptr [rdi], 1
```

4. 操作数位宽要匹配。

```asm
# 常用修正：读 32 位就用 eax
mov eax, dword ptr [rdi]
```

5. 传统高 8 位寄存器 `ah/bh/ch/dh` 不要和 `r8b` 这类新增 8 位寄存器混用。

### 1.6 8086 逻辑地址转物理地址

录音点名：逻辑地址转真实地址计算。

逻辑地址：段地址:偏移地址

```text
物理地址 = 段地址 * 16 + 偏移地址
        = 段地址 << 4 + 偏移地址
```

例：

```text
0x1460:0x0100 -> 0x14600 + 0x0100 = 0x14700
DS=0x1400, 偏移 0x0100 -> 0x14100
```

实模式速记：原始寻址模式

- 物理地址空间最大 1MB。
- 一个段最大 64KB。
- 一个物理地址可以有多个逻辑地址表示。

### 1.7 小端存储

x86-64 小端：

```text
低位字节放低地址，高位字节放高地址
```

例：

```asm
.long 0x12345678 #0x十六进制
```

内存从低地址到高地址：

```text
78 56 34 12
```

例：

```asm
array: .word 1, 2
mov eax, dword ptr array[rip]
```

内存：

```text
01 00 02 00
```

所以：

```text
EAX = 0x00020001
```

### 1.8 条件跳转

`cmp a, b`：

```text
偷偷算 a - b，只改标志，不保存结果
```

**无a有g**

无符号看 `above/below`：

| C | 跳转 |
|---|---|
| `>` | `ja` |
| `>=` | `jae` / `jnb` |
| `<` | `jb` / `jc` |
| `<=` | `jbe` |

有符号看 `greater/less`：

| C | 跳转 |
|---|---|
| `>` | `jg` |
| `>=` | `jge` |
| `<` | `jl` |
| `<=` | `jle` |

相等：

```text
je/jz：ZF=1
jne/jnz：ZF=0
```

常错例：

```asm
mov al, 0xff
cmp al, 1
```

- 按无符号：`0xff = 255`，所以 `ja` 成立。
- 按有符号：`0xff = -1`，所以 `jl` 成立。

## 2. 分析题精简背诵

录音说：分析大题分值高，共 4 道，3 道结合 C 语言。

### 2.1 C 与汇编互推

看汇编推 C 的顺序：

1. 看参数在哪些寄存器。
2. 看返回值怎么放到 `eax/rax`。
3. 看 `cmp/test + jcc` 判断条件。
4. 看循环变量初始化、循环条件、更新语句。
5. 看数组地址：`[base+index*scale]`。

Linux x86-64 参数：

```text
RDI, RSI, RDX, RCX, R8, R9
```

32 位 `int` 常看：

```text
EDI, ESI, EDX, ECX, R8D, R9D
```

返回值：

```text
int -> EAX
long long / 指针 -> RAX
```

例：

```asm
add:
    mov eax, edi
    add eax, esi
    ret
```

等价：

```c
int add(int a, int b)
{
    return a + b;
}
```

### 2.2 list 列表文件必考

录音原意：往年高频失分，今年必考。

必会三件事：

1. 根据 `.long/.word/.byte/.quad` 填数据字节。
2. 根据小端规则写机器码/数据字节顺序。
3. 根据 `.align` 计算偏移地址和填充字节。

做题步骤：

```text
先写每个数据的字节数
再按小端拆字节
再看 .align 是否需要补齐
最后逐项累计偏移地址
```

`.align n` 常见理解：

```text
把当前位置补到 n 的整数倍地址
```

例：

```text
当前位置 0x0003，.align 4 -> 补到 0x0004，需要补 1 字节
当前位置 0x0005，.align 4 -> 补到 0x0008，需要补 3 字节
```

### 2.3 乘除法分析

除法寄存器搭配：

商后余前

| 位宽 | 被除数 | 除数 | 商 | 余数 |
|---|---|---|---|---|
| 16 位 | `DX:AX` | r/m16 | `AX` | `DX` |
| 32 位 | `EDX:EAX` | r/m32 | `EAX` | `EDX` |
| 64 位 | `RDX:RAX` | r/m64 | `RAX` | `RDX` |

无符号 32 位除法：

```asm
xor edx, edx
div ebx
```

有符号 32 位除法：

```asm
cdq
idiv ebx
```

有符号 64 位除法：

```asm
cqo
idiv rbx
```

背：

```text
div/idiv 只有一个显式除数
商在前，余在后：EAX 商，EDX 余
idiv 余数符号跟被除数相同，商向 0 截断
```

分析公式时重点看：

```asm
imul eax, ebx
sub eax, ecx
cdq
idiv edi
```

对应：

```c
quot = (X * Y - V) / Z;
rem  = (X * Y - V) % Z;
```

### 2.4 移位 / 循环移位

**h逻辑啊算数**

**o无c，c有c**

| 指令 | 含义 | 移出位 |
|---|---|---|
| `shl/sal` | 左移，低位补 0 | 最高位进 CF |
| `shr` | 逻辑右移，高位补 0 | 最低位进 CF |
| `sar` | 算术右移，高位补符号位 | 最低位进 CF |
| `rol/ror` | 不经过 CF 的循环移位 | 位绕回 |
| `rcl/rcr` | 经过 CF 的循环移位 | CF 参与循环 |

用途：

```text
shl n 约等于乘 2^n
shr n 约等于无符号除 2^n
sar n 约等于有符号除 2^n，但负数截断方向要小心
```

### 2.5 128 位大数加减

双 64 位表示：

```text
A = A_hi:A_lo
B = B_hi:B_lo
```

128 位加法：

```asm
add rax, rbx
adc rdx, rcx
```

128 位减法：

```asm
sub rax, rbx
sbb rdx, rcx
```

背：

```text
低位先算，高位用 CF 衔接
加法高位用 adc，减法高位用 sbb
```

### 2.6 子程序阅读

`call`：

```text
push 返回地址 + jmp 子程序
```

`ret`：

```text
pop 返回地址到 RIP
```

栈：

```text
push rax -> rsp -= 8, [rsp] = rax
pop rdx  -> rdx = [rsp], rsp += 8
```

子程序题必看：

1. 参数来源。
2. 返回值放哪里。
3. 子程序有没有保护/破坏寄存器。
4. `call` 后返回到哪一条指令。
5. 主程序和子程序整体功能。

## 3. 编程题精简背诵

录音说：编程大题 2 道，综合性强。

### 3.1 编程题硬性得分点

必须写完整程序：

```asm
.intel_syntax noprefix
.data
    # 数据定义

.text
.globl main
main:
    # 主程序
    mov eax, 0
    ret

sub:
    # 子程序
    ret
```

不能只写核心代码。题目要求有主程序、子程序、宏，就都要写出来。

除法题要写异常边界：

```asm
test edi, edi
jz div_zero
```

### 3.2 宏汇编必背

宏是汇编时展开，不是运行时调用。

```asm
.macro BINOP op, dst, src
    \op \dst, \src
.endm

BINOP add, eax, ebx
BINOP xor, ecx, edx
```

展开后：

```asm
add eax, ebx
xor ecx, edx
```

宏 vs 子程序：

| 宏 | 子程序 |
|---|---|
| 汇编时展开 | 运行时调用 |
| 不用 `call/ret` | 用 `call/ret` |
| 每次调用展开一份 | 通常只有一份代码 |
| 无调用返回开销 | 有调用返回开销 |

### 3.3 循环 + 分支模板

数组遍历：

```asm
    xor eax, eax        # result = 0
    xor ecx, ecx        # i = 0
    test esi, esi
    jle .done

.L:
    mov edx, dword ptr [rdi+rcx*4]
    # body
    inc ecx
    cmp ecx, esi
    jl .L

.done:
    ret
```

判断正数：

```asm
cmp edx, 0
jle .not_pos
```

判断非零：

```asm
test edx, edx
jnz .nonzero
```

判断负数：

```asm
test eax, eax
js .negative
```

### 3.4 子程序模板

Linux x86-64：

```asm
.intel_syntax noprefix
.text
.globl add
add:
    mov eax, edi
    add eax, esi
    ret
```

Win64 识别：

```text
RCX, RDX, R8, R9
调用 API 注意 32 字节影子空间、16 字节栈对齐
```

### 3.5 C 与汇编互写常用结构

`if (a > b)` 有符号：

```asm
cmp edi, esi
jg .then
```

`if ((unsigned)a > (unsigned)b)` 无符号：

```asm
cmp edi, esi
ja .then
```

`while (i < n)`：

```asm
cmp ecx, esi
jl .loop
```

`return a[i]`：

```asm
mov eax, dword ptr [rdi+rcx*4]
ret
```

## 4. 临考最后 30 分钟只背这些

1. `list`：小端、`.long/.word/.byte/.quad`、`.align`、偏移累计。
2. 跳转：无符号 `ja/jb`，有符号 `jg/jl`。
3. 除法：`EDX:EAX / r/m32 -> EAX 商，EDX 余`；有符号前 `cdq`，无符号前 `xor edx, edx`。
4. 128 位：`add+adc`，`sub+sbb`。
5. 子程序：`call` 压返回地址，`ret` 弹回；Linux 参数 `RDI RSI RDX RCX R8 R9`。
6. 宏：`.macro/.endm`，汇编时展开，不用 `call/ret`。
7. 编程题：完整框架、主程序、子程序、宏、循环分支、除数为 0。
8. 选择题：第一章概念、x86/ARM 对比、补码、标识符、指令错误、8086 物理地址。
