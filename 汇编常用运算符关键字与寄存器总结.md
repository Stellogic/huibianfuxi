# 汇编常用运算符关键字与寄存器总结

口径：当前课程主线按 `x86-64 + GAS + Intel syntax`。旧题里的 8086、ARM/鲲鹏只放在最后做识别。

---

## 0. 最短背诵版

1. `[]` 表示访问内存，`lea` 只算地址，不访问内存。
2. `byte/word/dword/qword ptr` 指明内存操作数大小。
3. 全局变量优先写 `label[rip]`，取地址常写 `lea rax, label[rip]`。
4. 双操作数指令大多不能两个都是内存，大小要一致，立即数不能当目的操作数。
5. `cmp` 像 `sub`，只改标志；`test` 像 `and`，只改标志。
6. 无符号比较用 `above/below`：`ja/jb`；有符号比较用 `greater/less`：`jg/jl`。
7. `loop` 默认用 `ECX/RCX`，会自动减 1，不影响标志。
8. 除法固定：商在 `A`，余数在 `D`，32 位是 `EDX:EAX / r/m32`。
9. Linux 参数：`RDI RSI RDX RCX R8 R9`；Win64 参数：`RCX RDX R8 R9`。
10. 返回值通常在 `RAX/EAX`，栈顶指针是 `RSP`。

## 1. GAS Intel 语法符号和关键字

### 1.1 基本符号

| 写法 | 含义 | 例子 |
|---|---|---|
| `#` | 注释 | `mov eax, 1 # eax=1` |
| `label:` | 标号 | `main:`、`.L1:` |
| `[]` | 内存寻址 | `mov eax, [rdi]` |
| `+ - *` | 地址表达式运算 | `[rdi+rcx*4+8]` |
| `,` | 分隔操作数 | `add eax, ebx` |
| `0x` | 十六进制常量 | `0xff` |
| `'A'` | 字符常量 | `mov al, 'A'` |
| `\n` | 字符串转义 | `.asciz "hi\n"` |

注意：当前课程里十六进制数优先写 `0xFF`，不要写成 MASM 风格的 `0FFH`。

### 1.2 内存大小关键字

| 关键字 | 大小 | 常见用途 |
|---|---:|---|
| `byte ptr` | 1 字节 | `char`、单字符、字节数组 |
| `word ptr` | 2 字节 | 16 位数据 |
| `dword ptr` | 4 字节 | `int`、32 位数据 |
| `qword ptr` | 8 字节 | `long long`、指针、64 位数据 |

例：

```asm
mov byte ptr [rdi], 1
mov dword ptr [rdi], 100
mov rax, qword ptr [rsi]
```

必须写大小的常见场景：

```asm
mov [rdi], 1              # 错：不知道写几字节
mov dword ptr [rdi], 1    # 对
```

### 1.3 地址表达式

通式：

```asm
[base + index * scale + displacement]
```

含义：

```text
有效地址 = 基址寄存器 + 变址寄存器 * 比例因子 + 位移量
```

比例因子只能是：

```text
1, 2, 4, 8
```

常见数组地址：

| C 写法 | 汇编地址 |
|---|---|
| `char s[i]` | `[rdi+rcx]` |
| `int a[i]` | `[rdi+rcx*4]` |
| `long long b[i]` | `[rdi+rcx*8]` |
| `a[i+2]` | `[rdi+rcx*4+8]` |

RIP 相对寻址：

```asm
mov eax, dword ptr x[rip]     # 读全局变量 x
mov dword ptr x[rip], eax     # 写全局变量 x
lea rdi, msg[rip]             # 取 msg 的地址
```

`lea` 重点：

```asm
lea rax, [rdi+rcx*4]       # rax = rdi + rcx*4，不读内存
lea eax, [rdi+rdi*8+7]     # eax = 9*rdi + 7
```

### 1.4 数据定义伪指令

| 伪指令 | 含义 |
|---|---|
| `.byte` | 定义 1 字节数据 |
| `.word` | 定义 2 字节数据 |
| `.long` | 定义 4 字节数据 |
| `.quad` | 定义 8 字节数据 |
| `.ascii "abc"` | 字符串，不自动补 0 |
| `.asciz "abc"` / `.string "abc"` | 字符串，自动补 `\0` |
| `.space n` | 分配 n 字节空间 |
| `.space n, fill` | 分配 n 字节并用 fill 填充 |
| `.align n` | 对齐到 n 的整数倍地址 |
| `.equ name, value` | 定义不可改符号常量 |
| `.set name, value` | 定义可重新赋值符号 |

小端例子：

```asm
.long 0x12345678
```

内存从低地址到高地址：

```text
78 56 34 12
```

### 1.5 段、符号、模块伪指令

| 关键字 | 含义 |
|---|---|
| `.intel_syntax noprefix` | 使用 Intel 语法，不给寄存器加 `%` |
| `.data` | 数据段 |
| `.text` | 代码段 |
| `.bss` | 未初始化数据段 |
| `.rodata` | 只读数据段 |
| `.globl name` | 导出符号，供链接器/其他模块使用 |
| `.extern name` | 声明外部符号 |
| `.include "file.s"` | 源文本包含 |
| `.macro name args...` | 定义宏 |
| `.endm` | 结束宏定义 |

最小程序框架：

```asm
.intel_syntax noprefix
.text
.globl main
main:
    mov eax, 0
    ret
```

宏：

```asm
.macro BINOP op, dst, src
    \op \dst, \src
.endm

BINOP add, eax, ebx
```

展开后相当于：

```asm
add eax, ebx
```

## 2. 常用指令关键字

### 2.1 传送和地址

| 指令 | 作用 | 备注 |
|---|---|---|
| `mov dst, src` | 传送数据 | 不能普通内存到内存 |
| `lea dst, mem` | 计算地址表达式 | 不访问内存 |
| `xchg a, b` | 交换 | 不能和立即数交换 |
| `movzx dst, src` | 零扩展传送 | 无符号扩展 |
| `movsx dst, src` | 符号扩展传送 | 有符号扩展 |
| `movsxd dst, src32` | 32 位符号扩展到 64 位 | 常用于 int 到 long |

常错：

```asm
mov dword ptr [rdi], dword ptr [rsi]  # 错

mov eax, dword ptr [rsi]              # 对
mov dword ptr [rdi], eax
```

### 2.2 算术运算

| 指令 | 含义 | 常考点 |
|---|---|---|
| `add dst, src` | 加法 | 设置标志 |
| `adc dst, src` | 带进位加法 | 多精度加法高位 |
| `sub dst, src` | 减法 | 设置标志 |
| `sbb dst, src` | 带借位减法 | 多精度减法高位 |
| `inc dst` | 加 1 | 不影响 CF |
| `dec dst` | 减 1 | 不影响 CF |
| `neg dst` | 求相反数 | `0 - dst` |
| `cmp a, b` | 比较 | 偷算 `a-b`，不保存结果 |

128 位加减：

```asm
# A = A + B, A_lo=rax, A_hi=rdx, B_lo=rbx, B_hi=rcx
add rax, rbx
adc rdx, rcx

# A = A - B
sub rax, rbx
sbb rdx, rcx
```

### 2.3 乘除法

| 指令 | 含义 | 结果 |
|---|---|---|
| `imul eax, imm` | 有符号乘法，双操作数/三操作数形式常用 | 结果在目的寄存器 |
| `mul r/m32` | 无符号乘法 | `EDX:EAX = EAX * r/m32` |
| `imul r/m32` | 有符号乘法，单操作数形式 | `EDX:EAX = EAX * r/m32` |
| `div r/m32` | 无符号除法 | `EDX:EAX / r/m32` |
| `idiv r/m32` | 有符号除法 | `EDX:EAX / r/m32` |
| `cdq` | `EAX` 符号扩展到 `EDX:EAX` | 32 位有符号除法前 |
| `cqo` | `RAX` 符号扩展到 `RDX:RAX` | 64 位有符号除法前 |

除法固定搭配：

| 位宽 | 被除数 | 除数 | 商 | 余数 |
|---|---|---|---|---|
| 16 位 | `DX:AX` | r/m16 | `AX` | `DX` |
| 32 位 | `EDX:EAX` | r/m32 | `EAX` | `EDX` |
| 64 位 | `RDX:RAX` | r/m64 | `RAX` | `RDX` |

准备除法：

```asm
# 无符号 32 位
xor edx, edx
div ebx

# 有符号 32 位
cdq
idiv ebx

# 有符号 64 位
cqo
idiv rbx
```

### 2.4 逻辑和位操作

| 指令 | 含义 | 标志 |
|---|---|---|
| `and dst, src` | 按位与 | `CF=0, OF=0`，更新 `SF/ZF/PF` |
| `or dst, src` | 按位或 | `CF=0, OF=0`，更新 `SF/ZF/PF` |
| `xor dst, src` | 按位异或 | `CF=0, OF=0`，更新 `SF/ZF/PF` |
| `not dst` | 按位取反 | 不影响标志 |
| `test a, b` | 偷算 `a & b` | 只改标志，不保存结果 |
| `bt a, bit` | 取某一位到 CF | 常配合 `jc/jnc` |

口诀：

```text
AND：1 & x = x，0 & x = 0
OR ：1 | x = 1，0 | x = x
XOR：1 ^ x = ~x，0 ^ x = x
```

常用判断：

```asm
test eax, eax
jz  zero          # eax == 0
js  negative      # eax 最高位为 1

test al, 1
jnz odd           # al 最低位为 1

bt eax, 5
jc bit5_is_1
```

### 2.5 移位和循环移位

| 指令 | 含义 | 移出位 |
|---|---|---|
| `shl/sal dst, n` | 左移，低位补 0 | 最高位进 CF |
| `shr dst, n` | 逻辑右移，高位补 0 | 最低位进 CF |
| `sar dst, n` | 算术右移，高位补符号位 | 最低位进 CF |
| `rol dst, n` | 循环左移，不带 CF | 位绕回，末次移出位进 CF |
| `ror dst, n` | 循环右移，不带 CF | 位绕回，末次移出位进 CF |
| `rcl dst, n` | 带 CF 循环左移 | CF 参与循环 |
| `rcr dst, n` | 带 CF 循环右移 | CF 参与循环 |
| `shld dst, src, n` | 双精度左移 | src 的高位补入 dst 低位 |
| `shrd dst, src, n` | 双精度右移 | src 的低位补入 dst 高位 |

移位次数：

- 立即数可以直接写：`shl eax, 3`
- 变量移位次数通常放 `CL`：`shl eax, cl`

### 2.6 跳转和循环

`cmp a, b` 后按 `a-b` 判断。

无符号比较：

| C 条件 | 跳转 | 标志 |
|---|---|---|
| `a > b` | `ja` | `CF=0 且 ZF=0` |
| `a >= b` | `jae` / `jnb` | `CF=0` |
| `a < b` | `jb` / `jc` | `CF=1` |
| `a <= b` | `jbe` | `CF=1 或 ZF=1` |

有符号比较：

| C 条件 | 跳转 | 标志 |
|---|---|---|
| `a > b` | `jg` / `jnle` | `ZF=0 且 SF=OF` |
| `a >= b` | `jge` / `jnl` | `SF=OF` |
| `a < b` | `jl` / `jnge` | `SF!=OF` |
| `a <= b` | `jle` / `jng` | `ZF=1 或 SF!=OF` |

其他常用跳转：

| 指令 | 含义 |
|---|---|
| `jmp label` | 无条件跳转 |
| `je/jz` | 相等，`ZF=1` |
| `jne/jnz` | 不等，`ZF=0` |
| `jc` / `jnc` | `CF=1` / `CF=0` |
| `jo` / `jno` | `OF=1` / `OF=0` |
| `js` / `jns` | `SF=1` / `SF=0` |
| `jp` / `jnp` | `PF=1` / `PF=0` |
| `loop label` | `ECX/RCX` 减 1，非 0 则跳 |
| `loope/loopz` | 计数非 0 且 `ZF=1` 则跳 |
| `loopne/loopnz` | 计数非 0 且 `ZF=0` 则跳 |
| `jecxz label` | `ECX=0` 则跳 |
| `jrcxz label` | `RCX=0` 则跳 |

`loop` 近似：

```text
ecx = ecx - 1
if ecx != 0:
    jump label
```

注意：`loop` 不影响标志；`dec ecx` 会影响部分标志。

### 2.7 栈和子程序

| 指令 | 含义 |
|---|---|
| `push src` | 压栈，64 位下通常 `RSP -= 8` |
| `pop dst` | 出栈，64 位下通常 `RSP += 8` |
| `call target` | 压入返回地址，再跳到 target |
| `ret` | 弹出返回地址到 `RIP` |
| `ret n` | `ret` 后额外 `RSP += n` |
| `leave` | 恢复栈帧，近似 `mov rsp, rbp; pop rbp` |

`call` 和 `jmp` 区别：

```text
call 会保存返回地址；jmp 只跳转，不保存返回地址。
```

## 3. 标志位关键字

| 标志 | 全称 | 含义 |
|---|---|---|
| `CF` | Carry Flag | 无符号进位/借位 |
| `OF` | Overflow Flag | 有符号溢出 |
| `SF` | Sign Flag | 结果最高位为 1 |
| `ZF` | Zero Flag | 结果为 0 |
| `PF` | Parity Flag | 结果低 8 位中 1 的个数为偶数 |
| `DF` | Direction Flag | 字符串指令方向，0 递增，1 递减 |
| `IF` | Interrupt Flag | 中断允许标志 |

最常考：

```text
CF 看无符号，OF 看有符号，ZF 看是否为 0，SF 看符号位。
```

保存/恢复标志：

```asm
pushfq
pop rax

push rax
popfq
```

## 4. 寄存器用法总结

### 4.1 通用寄存器大小对应表

| 64 位 | 低 32 位 | 低 16 位 | 低 8 位 |
|---|---|---|---|
| `RAX` | `EAX` | `AX` | `AL/AH` |
| `RBX` | `EBX` | `BX` | `BL/BH` |
| `RCX` | `ECX` | `CX` | `CL/CH` |
| `RDX` | `EDX` | `DX` | `DL/DH` |
| `RSI` | `ESI` | `SI` | `SIL` |
| `RDI` | `EDI` | `DI` | `DIL` |
| `RBP` | `EBP` | `BP` | `BPL` |
| `RSP` | `ESP` | `SP` | `SPL` |
| `R8` | `R8D` | `R8W` | `R8B` |
| `R9` | `R9D` | `R9W` | `R9B` |
| `R10` | `R10D` | `R10W` | `R10B` |
| `R11` | `R11D` | `R11W` | `R11B` |
| `R12` | `R12D` | `R12W` | `R12B` |
| `R13` | `R13D` | `R13W` | `R13B` |
| `R14` | `R14D` | `R14W` | `R14B` |
| `R15` | `R15D` | `R15W` | `R15B` |

关键规则：

- 写 `EAX/EBX/...` 这类 32 位寄存器，通常会把对应 64 位寄存器高 32 位清 0。
- 写 `AX/AL` 只改低 16/8 位，不会自动清高位。
- `AH/BH/CH/DH` 是传统高 8 位，不能和 `SIL/DIL/BPL/SPL/R8B...` 在同一条需要 REX 的指令里混用。

### 4.2 通用寄存器用途速查

| 寄存器 | 常见用途 | 考场记忆 |
|---|---|---|
| `RAX/EAX/AX/AL` | 返回值、累加器、乘除法低位/商 | A 放结果、商 |
| `RBX/EBX/BX/BL` | 普通数据、基址、除数 | 可放数据，常保存值 |
| `RCX/ECX/CX/CL` | 循环计数、移位次数、参数 | C 管 count，`CL` 管移位次数 |
| `RDX/EDX/DX/DL` | 乘除法高位/余数、参数 | D 放余数 |
| `RSI/ESI/SI/SIL` | 源地址、Linux 第 2 参数 | SI = source index |
| `RDI/EDI/DI/DIL` | 目的地址、Linux 第 1 参数 | DI = destination index |
| `RBP/EBP/BP/BPL` | 栈帧基址 | base pointer |
| `RSP/ESP/SP/SPL` | 栈顶指针 | stack pointer |
| `R8-R15` | x86-64 新增通用寄存器 | 传参、临时值、寻址都可用 |

一句话：

```text
A 放结果，D 放余数，C 管循环和移位，SP 管栈顶，BP 管栈帧，SI/DI 管源和目的。
```

### 4.3 特殊寄存器

| 寄存器 | 用途 |
|---|---|
| `RIP` | 指令指针，保存下一条要执行的指令地址 |
| `RFLAGS` | 标志寄存器，保存 `CF/OF/SF/ZF/PF` 等 |
| `CS` | 代码段寄存器 |
| `DS` | 数据段寄存器 |
| `SS` | 栈段寄存器 |
| `ES` | 附加段寄存器 |
| `FS/GS` | 段寄存器，现代系统中常有特殊用途 |

当前课程主线中：

- 常见 `label[rip]`，即 RIP 相对寻址。
- `RFLAGS` 通过 `cmp/test/add/sub` 等指令更新，再由 `jcc` 判断。
- 段寄存器主要做选择题识别；64 位编程题一般不主动操作它们。

### 4.4 ABI 参数和返回值

Linux x86-64：

| 第几个整数/指针参数 | 寄存器 |
|---:|---|
| 1 | `RDI` |
| 2 | `RSI` |
| 3 | `RDX` |
| 4 | `RCX` |
| 5 | `R8` |
| 6 | `R9` |
| 返回值 | `RAX/EAX` |

32 位 `int` 参数看低 32 位：

```text
EDI, ESI, EDX, ECX, R8D, R9D
```

Win64：

| 第几个整数/指针参数 | 寄存器 |
|---:|---|
| 1 | `RCX` |
| 2 | `RDX` |
| 3 | `R8` |
| 4 | `R9` |
| 返回值 | `RAX/EAX` |

Win64 调 API 额外记：

- 调用者通常预留 32 字节 shadow space。
- 调用前后注意 16 字节栈对齐。

### 4.5 调用者保存和被调用者保存

考前够用口径：

```text
返回值寄存器不要保护后恢复，否则结果会被你自己覆盖。
入口参数寄存器通常也不是必须原样恢复。
子程序只保护 ABI 要求保持不变、且自己确实改动的寄存器。
```

Linux x86-64 常见：

| 类型 | 寄存器 |
|---|---|
| 调用者保存 | `RAX, RCX, RDX, RSI, RDI, R8-R11` |
| 被调用者保存 | `RBX, RBP, R12-R15` |
| 栈指针 | `RSP` 必须平衡恢复 |

Win64 常见：

| 类型 | 寄存器 |
|---|---|
| 调用者保存 | `RAX, RCX, RDX, R8-R11` |
| 被调用者保存 | `RBX, RBP, RDI, RSI, R12-R15` |
| 栈指针 | `RSP` 必须平衡恢复 |

如果试卷只考课程基础，优先背参数顺序和返回值；保存规则用于阅读较复杂子程序。

### 4.6 乘除、循环、栈的固定寄存器

乘除法：

| 场景 | 固定寄存器 |
|---|---|
| 32 位乘法低位 | `EAX` |
| 32 位乘法高位 | `EDX` |
| 32 位除法被除数 | `EDX:EAX` |
| 32 位除法商 | `EAX` |
| 32 位除法余数 | `EDX` |
| 64 位除法被除数 | `RDX:RAX` |
| 64 位除法商 | `RAX` |
| 64 位除法余数 | `RDX` |

循环和移位：

| 指令/场景 | 固定寄存器 |
|---|---|
| `loop` | `ECX/RCX` |
| `jecxz` | `ECX` |
| `jrcxz` | `RCX` |
| 变量移位次数 | `CL` |

栈和调用：

| 指令/场景 | 固定寄存器 |
|---|---|
| 栈顶 | `RSP` |
| 栈帧 | `RBP` |
| 当前执行位置 | `RIP` |
| 返回值 | `RAX/EAX` |

## 5. 常见错误速查

| 错误写法 | 错因 | 修正 |
|---|---|---|
| `mov [rdi], [rsi]` | 不能内存到内存 | 先读到寄存器再写 |
| `add 1, eax` | 立即数不能作目的操作数 | `add eax, 1` |
| `mov [rdi], 1` | 内存大小不明确 | `mov dword ptr [rdi], 1` |
| `mov eax, rbx` | 操作数大小不一致 | `mov rax, rbx` 或 `mov eax, ebx` |
| `mov al, 0x123` | 立即数超过 8 位 | 换更宽寄存器 |
| `mov ds, 100` | 段寄存器不能直接接立即数 | `mov ax,100; mov ds,ax` |
| `mov ah, r8b` | 高 8 位和新增低 8 位混用 | 换 `al/bl/cl/dl` 等 |
| `xchg eax, 1` | `xchg` 不能和立即数交换 | 先把立即数放寄存器 |
| `idiv ebx` 前没 `cdq` | 有符号被除数未准备 | `cdq; idiv ebx` |
| `div ebx` 前 `edx` 有旧值 | 无符号被除数错误 | `xor edx,edx; div ebx` |
| 有符号大于用 `ja` | `ja` 是无符号 | 用 `jg` |
| 无符号小于用 `jl` | `jl` 是有符号 | 用 `jb` |

## 6. 旧题/选择题识别补丁

### 6.1 8086 常见寄存器

| 类型 | 寄存器 |
|---|---|
| 通用 | `AX, BX, CX, DX` |
| 变址/指针 | `SI, DI, BP, SP` |
| 段寄存器 | `CS, DS, SS, ES` |
| 指令指针 | `IP` |
| 标志 | `FLAGS` |

实模式物理地址：

```text
物理地址 = 段地址 * 16 + 偏移地址
```

例：

```text
0x1460:0x0100 -> 0x14700
```

### 6.2 AArch64/鲲鹏常见寄存器

| 寄存器 | 用途 |
|---|---|
| `x0-x7` | 常见参数/返回值寄存器，`x0` 也常放返回值 |
| `x0-x5` | 本课程选择题常说系统调用前 6 个参数 |
| `sp` | 栈指针 |
| `pc` | 程序计数器概念 |
| `PSTATE` | 状态位逻辑统称 |
| `SPSR_ELn` | 异常发生时保存 `PSTATE` |
| `ELR_ELn` | 保存异常返回地址/旧 PC |

异常等级：

| 等级 | 含义 |
|---|---|
| `EL0` | 应用程序 |
| `EL1` | 操作系统内核 |
| `EL2` | 虚拟机监控程序 |
| `EL3` | 安全监控/TrustZone |

触发系统调用：

```asm
SVC #0
```
